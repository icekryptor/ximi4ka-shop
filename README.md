# ximi4ka-shop

E-commerce storefront and admin panel for Ximi4ka, replacing the current Tilda site. This repository is a sibling to the ERP repo at `../ximi4ka` and is organized as an npm workspaces monorepo containing `web/` (Next.js 15 App Router), `api/` (Node + Express + TypeORM), and `shared/` (shared TypeScript types). See the design doc at `../ximi4ka/docs/plans/2026-04-19-tilda-to-shop-migration-design.md` for background and architecture.
