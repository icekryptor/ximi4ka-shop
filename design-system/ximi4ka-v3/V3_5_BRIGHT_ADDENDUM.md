# v3.5 — «Лабораторный журнал, но ярче» (Bright Addendum)

**Status:** Approved 2026-07-03. Дополняет [LABORATORY_JOURNAL.md](./LABORATORY_JOURNAL.md), не заменяет его.
**Причина:** обратная связь владельца — «визуал ближе к текущему сайту на Тильде: яркие контейнеры,
больше визуала продуктов, интерактивные прикольные элементы».

---

## 1. Что сохраняется из v3 (не трогать)

- Типографика: Unbounded / Inter / JetBrains Mono. Никаких новых шрифтов.
- Cream `#F2EFE8` как базовый фон страниц, ink `#0A0A0A` как ночная поверхность.
- Структура страниц, сетки, NotebookHeader, eyebrow-паттерн (● NN.0 / Метка), моно-подписи.
- Ритм cream → ink → cream.
- Motion-дисциплина: CSS-first, лёгкий, `prefers-reduced-motion` уважается,
  никаких тяжёлых библиотек (цель Lighthouse ≥ 90).

## 2. Что меняется: снимается «бюджет ≤ 5 вхождений purple»

Бренд-фиолет `#836efe` и фирменный градиент Tilda-сайта переходят из категории
«точечный акцент» в категорию **рабочая поверхность**. Правило v3 §5.4
(«Forbidden: brand-purple background blocks, brand-purple gradients») отменяется.
Остальные пункты §5.4 (italic-слово, callout-строки, eyebrow-буллеты) продолжают работать.

## 3. Токены v3.5

```css
:root {
  /* Фирменный градиент Tilda — единственный канонический */
  --gradient-lj-bright: linear-gradient(135deg, rgba(141,103,255,1) 0%, rgba(200,86,255,1) 100%);
  /* Hover-состояние градиентных поверхностей: чуть глубже */
  --gradient-lj-bright-deep: linear-gradient(135deg, rgba(122,82,255,1) 0%, rgba(184,62,246,1) 100%);

  /* Крупное скругление ярких контейнеров (у Tilda 40–55px; здесь сдержаннее) */
  --radius-lj-bright:    32px;   /* карточки категорий, hero-панель */
  --radius-lj-bright-sm: 24px;   /* фото-плиты карточек товара */

  /* Elevation ярких поверхностей */
  --shadow-lj-bright: 0 16px 40px -12px rgba(131, 110, 254, 0.45),
                      0 6px 16px -6px  rgba(200, 86, 255, 0.25);

  /* Текст на градиенте */
  --color-lj-on-bright:      #ffffff;
  --color-lj-on-bright-mute: rgba(255, 255, 255, 0.78);
}
```

**Скругления.** Исключение из правила v3 §2.5 («sharp 90° everywhere»):
ЯРКИЕ градиентные контейнеры и фото-плиты товаров получают крупное скругление.
Всё остальное (Mendeleev-ячейки, ink-панели, таблицы, рамки иллюстраций) остаётся острым.
Скругление = сигнал «это продуктовая/навигационная поверхность».

## 4. Какие компоненты переводятся на яркие поверхности

| Компонент | Было (v3) | Стало (v3.5) |
|---|---|---|
| `CategoryTileLJ` | cream-shade плита, острые углы, ink-текст | градиент `--gradient-lj-bright`, radius 32px, белая типографика, белый молекул-мотив, hover-lift |
| `PreFooterCta` CTA | bone-outline пилюля | градиентная пилюля `.lj-cta-bright` |
| `Hero` primary CTA | ink-пилюля → brand-deep hover | градиентная пилюля `.lj-cta-bright` |
| `Hero` визуал | type-only | справа яркая градиентная панель с фото флагманского продукта (скрыта < lg) |
| `CtaBlock` (CMS-блоки) | ink-пилюля | градиентная пилюля |
| Checkout / Cart / CartDrawer основные CTA | ink-пилюля | градиентная пилюля |
| `ProductCard` бейдж «Хит» | плоский brand-пилл | градиентный пилл с тенью |
| `ProductCard` фото-плита | cream-shade, острые углы | белая (#fff) плита, radius 24px, фото доминирует |
| `ProductCard` цена | text-3xl Unbounded | text-4xl Unbounded (крупнее) |
| `Ticker` (ui) | orange `accent` surface (v2-рудимент) | новый surface `bright` = градиент; orange удалён |

**Вторичные CTA** (ghost/outline) остаются ink-пилюлями — иерархия: градиент = главное
действие страницы, ink = второстепенное. На одной странице не более двух градиентных CTA
в зоне видимости.

## 5. Интерактив (CSS-first)

1. **Hover-lift карточек** — утилита `.lj-lift`: `translateY(-6px)` + рост тени, 0.4s
   `--ease-out-quart`. Применяется к CategoryTileLJ и ProductCard. Под
   `prefers-reduced-motion: reduce` — отключено.
2. **Дрейф молекул** — `@keyframes lj-drift` (лёгкий translate/rotate ±8px, 14s
   ease-in-out infinite alternate). Применяется к декоративным MoleculeMotif в hero.
   Бензол сохраняет вращение (v3), дрейф добавляется деталям.
3. **Пузырьки при добавлении в корзину** — уже реализовано (`AddToCartBurst`,
   `.lj-bubble` / `.lj-add-flash`). Не дублировать; кнопка приводится к `.lj-cta-bright`,
   вспышка перекрашивается в градиентную гамму (остаётся brand-цвет — согласовано).
4. **Маркиз-тикер фактов** — на главной между hero и каталогом: `Ticker`
   surface="bright" c фактами `161 опыт · 62 реактива · доставка СДЭК · от 8 лет ·
   4,9 из 5`. Пауза на hover (уже в компоненте).
5. **«Элемент дня» в футере** — Mendeleev-ячейка в футере; по hover/focus чистым CSS
   раскрывается подпись «элемент дня: Название · №NN». Логика выбора —
   `lib/elementOfDay.ts`, детерминирована датой (день года по модулю списка),
   покрыта юнит-тестами. Без JS-обработчиков — только `:hover`/`:focus-within`.

## 6. Контраст и доступность

- **Белый на градиенте:** заголовки на градиенте — только Unbounded 700/900 крупным
  кеглем (≥ 24px / large text, WCAG large-text порог 3:1: белый на #8d67ff ≈ 3.9:1 — проходит).
  Мелкий текст на градиенте — только `--color-lj-on-bright` (чистый белый) и не ниже
  11px mono uppercase; вспомогательный текст `--color-lj-on-bright-mute` не мельче 12px.
- **Ink-текст (`#0A0A0A`) на градиенте запрещён.** Ink живёт только на cream/белом.
- Градиентные CTA: белый текст, `font-medium` mono uppercase — тот же паттерн, что
  ink-пилюли, меняется только поверхность.
- Фокус-кольца: `focus-visible` кольцо `--color-lj-brand-deep` на светлом,
  белое на градиенте.
- Все декоративные анимации (`lj-drift`, `lj-lift`, тикеры) отключаются в
  `prefers-reduced-motion: reduce`.

## 7. Что по-прежнему запрещено

- Градиент как фон **целых секций** body-текста (только контейнеры/панели/CTA/тикер).
- Ink → ink стык секций без cream-паузы (правило v3 §1 живо).
- Орфографический orange (`--color-accent` v2) — вычищается при касании кода.
- Тяжёлые анимационные библиотеки, three.js, canvas-частицы.
