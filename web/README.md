# @ximi4ka-shop/web

Public storefront for Ximi4ka — chemistry experiment kits.

Part of the [`ximi4ka-shop`](../README.md) monorepo. The root README has setup,
architecture and development commands.

## Quick start

```bash
npm install            # from the repo root
cp .env.example .env   # in web/, if not already done
npm run dev            # starts Next.js on port 3000
```

## Scripts

- `npm run dev` — start dev server
- `npm run build` — production build
- `npm run start` — serve the production build
- `npm run test` — Vitest suite
- `npm run typecheck` — `tsc --noEmit`
- `npm run lint` — ESLint
